Bab 152: Menyempurnakan Benih Api

"Benih Api?"

Xiao Yan sedikit tercengang setelah mendengar kata-kata Yao Lao. Dia bingung antara tertawa dan menangis ketika dia berkata, "Mengapa saya merasa bahwa kata-kata Anda menyarankan sesuatu seperti memelihara hewan di rumah, menunggunya menjadi gemuk dan kemudian menyembelihnya?"

"Hehe, prinsipnya sama." Yao Lao mengangguk sambil tersenyum.

Xiao Yan menganggukkan kepalanya dan tatapannya menatap Api Ungu kecil yang menggeliat di jarinya yang merupakan satu-satunya harapannya untuk mengembangkan Metode Qi-nya. Api Surgawi agak terlalu jauh baginya sementara Api Ungu dari Amethyst Essence ini hanya cocok untuk kondisi dan kekuatannya saat ini.

"Bagaimana cara memperbaikinya?" Xiao Yan menghela nafas, mengangkat kepalanya dan bertanya.

"Tenang, Api Ungu ini tidak memiliki pemilik sehingga tidak akan memberikan banyak perlawanan terhadapmu. Menyempurnakannya menjadi Biji Api seharusnya tidak melibatkan terlalu banyak bahaya. " Yao Lao berkata sambil mengulurkan jarinya dan meletakkannya di kepala Xiao Yan, "Ikuti instruksiku."

Merasakan sejumlah besar informasi yang dituangkan ke dalam pikirannya, Xiao Yan dengan hati-hati menikmatinya sebelum menganggukkan kepalanya. Dia secara bertahap menutup matanya dan menyilangkan kakinya, mengadopsi posisi latihannya. Dia mengamati matanya, hidungnya lalu hatinya saat pikirannya secara bertahap masuk jauh ke dalam tubuhnya.

Pikirannya beredar melalui Jalur Qi dan akhirnya tiba di pusaran di perut bagian bawahnya. Munculnya pusaran Dou Qi anehnya jauh lebih kecil dari sebelumnya. Namun, jika diamati dengan cermat, kandungan pusaran itu jauh lebih padat dan diaglomerasi lebih baik dari sebelumnya.

Pikiran Xiao Yan berputar di sekitar pusaran sekali. Ketika dia tidak melihat ada masalah, Xiao Yan mengikuti instruksi Yao Lao dan mulai bertindak.

Pikiran Xiao Yan berkeliling pusaran dan perlahan memasukinya. Pada saat yang sama, Persepsi Spiritualnya berulang kali mencari Api Ungu yang dia kendalikan.

Xiao Yan dengan hati-hati memindai bagian dalam pusaran. Beberapa saat kemudian, banyak utas energi berwarna ungu yang sangat kecil mulai muncul secara bertahap. Ketika energi berwarna ungu muncul, Xiao Yan buru-buru mengendalikan Persepsi Spiritualnya dan melingkupinya dengan cepat.

"Buka lubang kecil di pusaran yang bisa berisi Api Ungu ..."

Mengingat kata-kata Yao Lao di benaknya, Xiao Yan dengan cepat mengepung pusaran itu dengan Persepsi Spiritualnya. Setelah keheningan singkat, Persepsi Spiritualnya membuat pusaran berputar dengan kecepatan yang sangat cepat.

Mengikuti pusaran yang berputar dengan kecepatan tinggi, Dou Qi yang mengalir di Jalur Qi-nya juga melonjak dengan cepat dan ganas. Namun, Xiao Yan tidak memiliki waktu luang untuk mengganggu mereka; pikirannya sepenuhnya terfokus pada ruang kosong kecil di tengah pusaran yang terbentuk dari rotasi cepat.

Melihat bahwa tindakannya efektif, Xiao Yan meningkatkan kekuatan Persepsi Spiritualnya yang mengelilingi pusaran. Mengikuti peningkatan kekuatan kesadaran spiritual, kecepatan pusaran berputar semakin cepat. Pada akhirnya, orbit pusaran berwarna kuning pucat tidak lagi terlihat sementara suara angin tak berbentuk bergema di luar pusaran. Pikiran Xiao Yan terus mengamati bagian tengah pusaran di mana ruang kosong semakin lebar. Ketika itu kira-kira seukuran kepalan tangan, Xiao Yan akhirnya menghela nafas lega dan perlahan menarik Persepsi Spiritualnya dari pusaran.

Setelah Persepsi Spiritual sepenuhnya ditarik, kecepatan rotasi pusaran juga mulai secara bertahap menjadi lebih lemah. Adapun lubang di tengah pusaran, ia mampu membentuk keseimbangan aneh dengan sekitarnya di bawah kendali pikiran Xiao Yan dan tidak menghilang dengan kecepatan putaran pusaran yang lebih lambat.

Melihat ruang kosong itu tidak hilang, Xiao Yan menghela nafas lagi di dalam hatinya. Pikirannya perlahan-lahan mengelilingi seutas benang energi berwarna ungu dan membimbingnya melalui blokade Dou Qi di sekitarnya sebelum memasukkannya ke dalam ruang kosong kecil itu.

Segera setelah energi berwarna ungu memasuki lubang kosong, Xiao Yan merasakan seluruh pusaran bergetar dan dia buru-buru menggunakan Persepsi Spiritualnya untuk menstabilkan pusaran.

Dengan Persepsi Spiritual Xiao Yan berusaha keras untuk mempertahankan situasi, pusaran akhirnya menjadi stabil. Dari dalam lubang tengah pusaran, Xiao Yan bisa melihat bahwa banyak benang jenis api Dou Qi terus menerus keluar dari pusaran dan mengalir ke energi berwarna ungu yang menempati lubang kosong.

Pikiran Xiao Yan dengan cemas mengamati energi kecil berwarna ungu. Saat semakin banyak jenis api Dou Qi dituangkan ke dalamnya, energi berwarna ungu mengeluarkan tangisan lembut. Sesaat kemudian, itu mulai berubah di bawah pengamatan pikiran Xiao Yan.

Warna energi berwarna ungu menjadi lebih gelap. Ketika gelap ke tahap tertentu, energi berwarna ungu tiba-tiba bergetar dan nyala api samar muncul dari energi berwarna ungu. Seketika, energi berwarna ungu dikonsumsi oleh api ungu kecil.

Di lubang kosong yang gelap, nyala api berwarna ungu berkedip sendirian. Kehangatan samarnya memuntahkan dan memasuki pusaran Dou Qi di luar.

Ketika panas samar memasuki pusaran Dou Qi, Xiao Yan bisa merasakan bahwa putaran pusaran menjadi lebih cepat. Reaksi awal Xiao Yan terhadap kejadian ini adalah keterkejutan dan langsung digantikan oleh kegembiraan yang tidak terkendali. Jika pusaran bisa berputar lebih cepat dengan sendirinya, itu berarti Xiao Yan tidak perlu mengendalikannya secara manual. Dou Qi dari lingkungan sekitarnya juga akan secara otomatis dan terus menerus dituangkan ke dalam tubuhnya, memungkinkan dia untuk terus-menerus dicuci oleh Dou Qi. Manfaat semacam ini tidak diragukan lagi sangat berguna untuk pelatihannya.

Setelah nyala api berwarna ungu muncul di lubang kosong, seolah-olah itu memulai mesin sirkulasi besar yang perlahan mulai berputar.

Api berwarna ungu diperlukan untuk menyerap jenis api Dou Qi dari dalam pusaran untuk tumbuh sementara panas yang dipancarkannya, meningkatkan tingkat penyerapan Dou Qi dari pusaran. Hubungan yang terus berulang ini secara signifikan menguntungkan Api Ungu dan pusaran.

Pikiran Xiao Yan mengamati pusaran itu sekali lagi. Ketika dia yakin tidak ada masalah, Xiao Yan akhirnya menarik pikirannya dari dalam tubuhnya. Dia secara bertahap membuka matanya dan cahaya ungu melintas di matanya yang hitam pekat. Sesaat kemudian, sepasang matanya secara bertahap memulihkan warna hitam pekat aslinya.

"Bagaimana itu? Berhasil?" Melihat Xiao Yan membuka matanya, Yao Lao yang menunggu dengan sabar bertanya.

"Ya, saya telah menyempurnakannya menjadi Benih Api." Xiao Yan mengangguk dan mengulurkan telapak tangannya. Dia menggosok jarinya dengan lembut. Setelah suara yang jelas, api kecil berwarna ungu muncul dari ujung jari Xiao Yan.

Dengan tatapan penuh rasa ingin tahu, Xiao Yan bermain dengan api ungu miliknya. Sesaat kemudian, jarinya dengan lembut menyentuh dinding gunung di sampingnya. Seketika, dinding itu hangus.

"Benda ini tampaknya jauh lebih lemah dari Api Dingin Tulang Guru ..." Xiao Yan tersenyum pahit sambil menatap tanda yang diciptakan oleh api.

"Tentu saja. Api Dingin Tulang Saya adalah Api Surgawi dan berada di peringkat kesebelas dalam Daftar Peringkat Api Surgawi. Jika barang rusak milikmu ini dapat dibandingkan dengannya, mengapa aku harus repot-repot mengambil risiko terbakar menjadi abu untuk mencarinya?" Mendengar komentar Xiao Yan, Yao Lao langsung memutar bola matanya dan memarahinya sambil tersenyum.

Xiao Yan tak berdaya menggelengkan kepalanya. Kata-kata Yao Lao itu merupakan pukulan telak baginya. Sesuatu yang untungnya dia dapatkan setelah melalui begitu banyak usaha digambarkan sebagai barang rusak oleh Yao Lao.

"Baiklah, berhentilah terlihat begitu tertindas. Api Ungu milikmu ini baru saja lahir, berapa banyak kekuatan yang dimilikinya? Setelah Anda menaikkannya, seharusnya tidak terlalu lemah. Apakah Anda tidak melihat kekuatan serangan Api Ungu oleh Singa Bersayap Amethyst? Bahkan wanita bernama Yun Zhi dengan kekuatan Dou Huang-nya tidak berani begitu saja memblokirnya. Dari sini, dapat dilihat bahwa nyala api ini tidak biasa." Melihat ekspresi kecewa Xiao Yan, Yao Lao berkata dengan tenang.

Xiao Yan menghela nafas dan mengangguk. Dengan senyum pahit, dia berkata, "Jika saya ingin meningkatkan Api Ungu kecil ini dengan kekuatan yang mirip dengan Singa Bersayap Amethyst, saya khawatir itu akan memakan waktu setidaknya beberapa dekade."

"Ini benar jika kita menggunakan penalaran normal." Yao Lao mengangguk. Ketika dia melihat ekspresi tertekan Xiao Yan, dia tanpa daya berkata, "Tapi apakah kamu tidak memiliki Amethyst Birth Essence? Benda itu adalah katalis terbaik dalam membangkitkan Api Ungu. Selama Anda dapat menahan sedikit rasa sakit yang ditimbulkannya kepada Anda, Api Ungu Anda seharusnya dapat tumbuh dengan kecepatan yang ekstrem. "

"Esensi Kelahiran Singa Amethyst?"

Mendengar ini, Xiao Yan tertegun sejenak. Kebahagiaan yang tak terkendali dengan cepat memenuhi wajahnya saat dia dengan kikuk mengeluarkan botol giok kecil dari cincin penyimpanan. Dia membuka tutupnya dan aroma hangat yang tidak biasa terpancar darinya.

Setelah menghirup dalam-dalam aroma yang dipenuhi dengan energi api yang padat ini, Xiao Yan tiba-tiba menyadari bahwa api ungu di jarinya juga menjadi lebih kuat di bawah aroma itu.

"Sungguh energi api yang padat." Xiao Yan tanpa sadar tersentak saat merasakan gerakan Api Ungu.

"Ketika Anda berlatih di masa depan, akan lebih baik jika Anda memilih untuk melakukannya di bawah terik matahari. Selain itu, Anda tidak dapat mengonsumsi terlalu banyak Amethyst Lion Birth Essence. Tentu saja, jika Anda ingin mengulangi pengalaman menyakitkan seperti terakhir kali, Anda dapat mengabaikan kata-kata saya. Yao Lao dengan bercanda berkata.

Mengingat rasa sakit dari tubuhnya yang terbakar terakhir kali, tubuh Xiao Yan bergidik. Perasaan itu bukanlah sesuatu yang ingin dia alami untuk kedua kalinya.

"Dengan kekuatanmu saat ini, kamu hanya boleh menggunakan jarimu untuk mencelupkannya sedikit dan menelannya ke dalam tubuhmu saat kamu berlatih. Jika Anda menggunakan terlalu banyak, Anda mungkin akan membakar diri Anda lagi." Yao Lao dengan serius mengingatkan.

"Ah." Xiao Yan dengan cepat menganggukkan kepalanya. Dia tidak berani membuat lelucon tentang ini; itu lebih dari cukup untuk mengalami rasa sakit seperti itu sekali.

"Menggunakan Amethyst Lion Birth Essence untuk mempercepat pertumbuhan Api Ungu... berapa lama sebelum bisa ditelan?" Xiao Yan dengan erat memegang botol giok di tangannya, mengangkat kepalanya dan dengan getir bertanya, "Beberapa tahun? Jika saya memiliki waktu seperti itu, saya mungkin juga pergi dan mencari Api Surgawi. "

"Dengan Amethyst Essence, kamu tidak membutuhkan banyak waktu. Selain itu, Api Ungu tidak perlu mencapai kekuatan yang dicapai Singa Bersayap Amethyst. Dengan kekuatanmu saat ini, jika kamu benar-benar merawat Api Ungu sejauh itu, kamu mungkin akan terbakar menjadi abu olehnya. Jika kamu tidak memiliki kekuatan yang cukup, bermain api akan membuatmu terbakar." Yao Lao menggelengkan kepalanya dan menyuarakan pikirannya, "Mungkin, sebulan dari sekarang, kamu harus bisa menggunakan 'Mantra Api' untuk mencoba dan menelan Api Ungu."

"Sebulan?" Xiao Yan menjilat bibirnya dan bergumam pelan. sebelum dengan cepat menganggukkan kepalanya dengan serius.

"Baiklah, sebulan dari sekarang, aku akan menelan Api Ungu dan mengembangkan Mantra Apiku!"
