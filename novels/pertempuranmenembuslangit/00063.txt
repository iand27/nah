Bab 63: Peringkat Api Surgawi

Di meja makan, Xiao Yan melirik ke arah pemakan ganas yang duduk di depannya. Saat Xiao Yu makan, gigi peraknya membuat suara berderak. Meringkuk bibirnya, Xiao Yan mengingat perasaan aneh namun indah sebelumnya dan jari-jari tangan kanannya tidak bisa membantu tetapi membelai cekungan telapak tangannya.

Duduk di seberangnya, Xiao Yu, yang selama ini menatap tajam ke arah Xiao Yan, melihat gerakannya dan wajahnya yang menawan memerah lagi.

Menatap aneh pada Xiao Yu yang menggertakkan giginya, dan kemudian melihat sikap acuh tak acuh Xiao Yan di dekatnya, Xun Er mengernyitkan alisnya karena curiga. Segera setelah itu, dia menggelengkan kepalanya tanpa daya, dan sedikit demi sedikit menelan makanan di depannya.

Tatapan Xiao Yan melewati Xiao Yu dan berhenti pada tubuh Xiao Ning. Pada saat ini, wajah Xiao Ning dipenuhi dengan kegembiraan; lekukan di mulutnya hampir memanjang di tengah wajahnya. Jari-jari ringan mengetuk tepi meja, Xiao Yan, menikmati kemalangan orang lain berpikir dalam hati: "Orang ini seharusnya sudah menerima Foundation Elixir, kan? Sedikit yang dia tahu, ramuan itu tidak memiliki banyak efek bagi mereka yang berada di Duan Qi ke-8 atau lebih..."

Tertawa pelan untuk dirinya sendiri, garis pandang Xiao Yan menyapu lingkungan dengan cara yang agak bosan. Melihat ayahnya yang tersenyum, hatinya bingung: "Makan bersama keluarga, bukankah hanya diadakan pada hari libur tertentu? Apa yang begitu meriah untuk hari ini? Mungkinkah menghabiskan banyak uang untuk membeli Foundation Elixir adalah acara yang layak untuk dirayakan?"

Membiarkan imajinasinya menjadi liar, Xiao Yan secara alami tidak tahu bahwa makan bersama keluarga ini adalah karena kata-kata yang diucapkan oleh orang berjubah hitam misterius tentang berkolaborasi dengan Xiao Clan jika ada kesempatan untuk melakukannya.

Bekerja sama dengan alkemis tingkat kedua atau lebih tinggi akan memberi mereka keuntungan yang akan membuat mata kebanyakan orang merah karena iri. Bahkan ada kemungkinan bahwa keluarga tersebut dapat naik peringkat, naik lebih tinggi dari dua keluarga besar lainnya. Jadi, tidak mengherankan bahwa ayah yang stabil dan tenang yang biasa akan sangat bahagia. Dan untuk beberapa tetua lainnya, mereka bahkan lebih, menyeringai dari telinga ke telinga. Melihat mata tua mereka yang setengah tertutup, terbukti bahwa nasib baik yang turun dari langit ini membuat mereka agak linglung.

Suasana meriah di jamuan makan bersama perlahan memudar dan setelah melihat gelombang perpisahan ayahnya, Xiao Yan segera melompat dari kursinya. Dia kemudian bergegas keluar dari aula dan langsung menuju kamarnya.

Tidak lama setelah Xiao Yan pergi, Xiao Yu mengatupkan giginya dan mengejarnya tetapi tidak dapat menemukan satu bayangan pun. Tak berdaya, dia hanya bisa menghentakkan kakinya dengan kebencian dan pergi dengan hati yang dipenuhi amarah.

......

Kembali ke kamarnya dan setelah belajar dari pengalaman pribadinya sendiri, Xiao Yan belum meminta Lao Yao untuk segera memperbaiki obatnya. Setelah menutup pintu dan jendela, dia dengan malas berguling ke tempat tidurnya dan merasa pusing, tertidur.

Larut malam, setelah semua orang tertidur, berbaring di tempat tidur, Xiao Yan membuka kedua matanya. Dia dengan bersemangat melompat dari tempat tidur dan mengeluarkan bahan-bahan yang tersembunyi dari lemari. Dengan hati-hati meletakkannya di atas meja, dia menoleh. Melihat ke arah Yao Lao, hantu yang melayang satu kaki dari tanah, Xiao Yan dengan lembut bertanya: "Guru, Anda siap sekarang, kan?"

"Akhirnya Anda telah belajar bagaimana menjadi sedikit berhati-hati; pemurnian pil membutuhkan lingkungan yang sangat tenang. Jika saya terganggu, konsekuensinya sangat parah. Saat ini, saya tidak akan merasakan kerusakan pembalasan, tetapi nanti ketika Anda mulai belajar alkimia, jika Anda terus ceroboh, maka saya khawatir Anda akan kehilangan nyawa kecil Anda sedikit lebih cepat daripada nanti. Yao Lao berjalan ke satu sisi meja dan dengan lembut menyentuh setiap bahan dengan telapak tangannya. Mengangguk kepalanya sedikit, nada suaranya yang membosankan membawa sedikit keparahan.

Menurunkan kepalanya karena malu, Xiao Yan menggaruk kepalanya saat menerima instruksi.

Hanya setelah melihat kepatuhan Xiao Yan, Yao Lao menghela nafas lega. Dia dengan muram merentangkan telapak tangannya dan nyala api putih meledak.

Dengan persepsi jiwanya yang terus-menerus mengendalikan suhu api, Yao Lao memanfaatkan waktu idle dan melirik Xiao Yan yang sedang menatap api di telapak tangan Yao Lao dengan rasa ingin tahu. Sedikit ragu-ragu, Yao Lao berbisik pelan: "Jajaran sebagian besar alkemis dapat dibedakan dari warna nyala api mereka."

"Alkemis biasa memiliki api dengan warna kuning samar. Saat peringkat mereka meningkat, warna api menjadi lebih gelap dan kekuatan api semakin kuat."

Setelah mendengar apa yang dikatakan, Xiao Yan mengedipkan mata, menunjuk api di tangan Yao Lao dan bertanya: "Kalau begitu guru, mengapa milikmu putih?"

"Haha, apa yang aku katakan sebelumnya adalah untuk api alkemis yang biasa terlihat. Namun di dalam lingkaran alkemis, selain mengandalkan Dou Qi sebagai katalis untuk nyala api, ada juga metode lain...." Yao Lao tersenyum dengan sedikit kebanggaan.

"Artinya, meminjam api!"

"Meminjam api?" Itu bukan kata-kata asing. Dia mengerti apa arti setiap kata, tapi Xiao Yan bingung. Bagaimana seseorang bisa meminjam api yang dibutuhkan untuk memurnikan pil?

"Itu benar, meminjam api." Mengangguk kepalanya, Yao Lao terkekeh: "Di dalam dunia tanpa batas ini, ada beberapa Api Surgawi langit dan bumi. Mungkin nyala api itu berasal dari dalam meteor yang jatuh dari langit atau mungkin nyala api itu berasal dari kedalaman gunung berapi, yang ditempa selama ratusan ribu tahun oleh lava... kekuatan Api Surgawi ini berkali-kali lipat lebih besar daripada kekuatannya. api yang diciptakan melalui Dou Qi. Memurnikan pil dengan api ini bahkan dapat meningkatkan efektivitas obat. Hanya saja, Api Surgawi langit dan bumi ini sangat ganas dan jarang datang. Dan bahkan jika Anda melihatnya, itu sangat sulit untuk dikendalikan."

"Banyak alkemis menghabiskan seluruh hidup mereka mencari Api Surgawi ini dan masih gagal mendapatkannya. Lagi pula, untuk mengendalikan Api Surgawi, Anda perlu membawa api ke dalam tubuh Anda; namun, sifat nyala api itu sangat merusak. Bahkan Berlian Emas Ajaib yang merupakan salah satu logam terkeras, tidak mampu menahan panas Api Surgawi dan tubuh manusia Anda yang rapuh.... Anda akan membakar diri Anda hidup-hidup. Oleh karena itu, hanya sedikit pemenang yang beruntung yang dapat, secara kebetulan, memperbaiki sejumlah kecil Api Surgawi. Begitu nyala api bergabung dengan tubuh, orang-orang seperti ini, tanpa kecuali adalah orang-orang dengan bakat luar biasa di dunia alkemis."

Tertegun untuk waktu yang lama, Xiao Yan menjilat bibirnya, tatapannya mengamati api putih di telapak tangan Yao Lao dan samar-samar bisa merasakan sensasi sedingin es.

"Api Guru pasti sejenis Api Surgawi, kan?" Xiao Yan bertanya.

"Hehe." Mengangkat nyala apinya, wajah Yao Lao memancarkan cahaya terang. Dengan mata berkobar, dia berbicara: "Dalam masyarakat alkemis Dou Qi Daratan, Api Surgawi yang dikenal berada di peringkat dalam "Peringkat Api Surgawi". Ada total 23 jenis dan nyala api saya berada di peringkat 11, yang dikenal sebagai "Api Dingin Tulang". Jenis Api Surgawi ini hanya muncul setiap seratus tahun, ketika gerhana terjadi di persimpangan Yin yang sangat dingin dan ekstrim ...... "

"Api Dingin Tulang?"

Tanpa berkedip, Xiao Yan menatap nyala api putih itu dan dengan lembut bergumam pada dirinya sendiri.
